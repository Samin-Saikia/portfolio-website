# 🌐 Modern Portfolio Website

A modern, animated, and fully responsive personal portfolio website built to showcase my work, skills, and professional profile.  
Designed with a premium dark-blue aesthetic, smooth animations, and interactive UI elements to create a memorable user experience.

---

## ✨ Features

- 🎨 Modern dark-blue premium UI
- ⚡ Smooth reveal-on-scroll animations
- 🖱️ Custom cursor glow with precise tracking
- 🔘 Advanced button hover animations (lift + color shift + icon motion)
- 🖼️ Framed hero and about section images
- 📱 Fully responsive for all screen sizes
- 🧩 Modular and clean code structure
- 🚀 Fast loading and optimized assets

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Modern layouts, animations, transitions)
- JavaScript (Vanilla)

### Tools & Platforms
- VS Code
- Git & GitHub
- Windows
